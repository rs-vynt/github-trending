# 🚀 Kong/insomnia

## 🌟 Trọng tâm
Insomnia là một API client mã nguồn mở, đa nền tảng mạnh mẽ hỗ trợ đa dạng các giao thức như GraphQL, REST, WebSockets, Server-Sent Events (SSE), và gRPC. Nó cung cấp môi trường toàn diện cho việc gỡ lỗi, thiết kế, kiểm thử, và giả lập (mock) các API với tính linh hoạt cao trong việc lưu trữ dữ liệu (Local, Git, hoặc Cloud).

## 🎯 Bài toán giải quyết & Khách hàng mục tiêu
- **Bài toán giải quyết:** Quá trình phát triển, kiểm thử và thiết kế API thường đòi hỏi nhiều công cụ rời rạc. Việc quản lý và đồng bộ dữ liệu API một cách an toàn giữa các thành viên trong nhóm cũng là một thách thức lớn.
- **Khách hàng mục tiêu:** Các lập trình viên (Developers), kỹ sư kiểm thử (QA Engineers), và các nhóm phát triển phần mềm cần một công cụ mạnh mẽ, trực quan để tương tác và quản lý vòng đời của API, đặc biệt là những người quan tâm đến việc lưu trữ dữ liệu an toàn (cục bộ hoặc Git).

## 🛠️ Công nghệ cốt lõi
- **Nền tảng:** Đa nền tảng (Mac, Windows, Linux) xây dựng dựa trên Electron.
- **Ngôn ngữ:** Node.js, JavaScript, TypeScript, React.
- **Giao thức hỗ trợ:** GraphQL, REST, WebSockets, SSE, gRPC.
- **Tính năng mở rộng:** Hỗ trợ OpenAPI, CLI (Inso) cho CI/CD, hệ thống Plugin.

## 💡 Tại sao đáng chú ý?
Insomnia nổi bật nhờ giao diện người dùng trực quan, sạch sẽ và tốc độ phản hồi nhanh. Điểm khác biệt lớn nhất là sự linh hoạt và tính bảo mật trong việc lưu trữ: người dùng có thể chọn lưu 100% dữ liệu cục bộ (Local Vault), đồng bộ qua Git độc lập không qua server của hãng, hoặc sử dụng Cloud Sync mã hóa đầu cuối (E2EE). Ngoài ra, Insomnia cung cấp hệ sinh thái plugin phong phú, giúp các nhà phát triển dễ dàng mở rộng và tùy biến công cụ theo nhu cầu.
