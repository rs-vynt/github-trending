# 🚀 Puppeteer

## 🌟 Trọng tâm
Cung cấp một thư viện JavaScript mạnh mẽ và dễ sử dụng với API cấp cao để điều khiển các trình duyệt web (như Chrome và Firefox) nhằm thực hiện các tác vụ tự động hóa một cách đáng tin cậy.

## 🎯 Bài toán giải quyết & Khách hàng mục tiêu
- **Bài toán giải quyết:** Tự động hóa các tác vụ trên trình duyệt như quét dữ liệu (web scraping), kiểm thử tự động (E2E testing), chụp ảnh màn hình, tạo tệp PDF từ trang web, và kiểm tra hiệu suất.
- **Khách hàng mục tiêu:** Các nhà phát triển web, kỹ sư kiểm thử (QA/Test engineers), và các nhà phân tích dữ liệu cần công cụ để tương tác và trích xuất dữ liệu từ các ứng dụng web.

## 🛠️ Công nghệ cốt lõi
- **Ngôn ngữ:** JavaScript/TypeScript.
- **Giao thức:** Chrome DevTools Protocol và WebDriver BiDi.
- **Hệ sinh thái:** Hoạt động trơn tru với Node.js, npm, và hỗ trợ các tính năng hiện đại thông qua MCP (Model Context Protocol).

## 💡 Tại sao đáng chú ý?
Puppeteer là tiêu chuẩn công nghiệp cho việc tự động hóa trình duyệt nhờ sự hỗ trợ chính thức từ Google, khả năng chạy ổn định ở chế độ headless (không cần giao diện), và API trực quan, dễ tích hợp vào bất kỳ dự án Node.js nào.
