# GhostTrack
Công cụ hữu ích để theo dõi vị trí hoặc số điện thoại di động, vì vậy công cụ này có thể được gọi là osint hoặc thu thập thông tin

<img src="https://github.com/HunxByts/GhostTrack/blob/main/asset/bn.png"/>

Bản cập nhật mới :
```Version 2.2```

### Cài đặt trên Linux (deb)
```
sudo apt-get install git
sudo apt-get install python3
```

### Cài đặt trên Termux
```
pkg install git
pkg install python3
```

### Cách sử dụng Công cụ
```
git clone https://github.com/HunxByts/GhostTrack.git
cd GhostTrack
pip3 install -r requirements.txt
python3 GhostTR.py
```
