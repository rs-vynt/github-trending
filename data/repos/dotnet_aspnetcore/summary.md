# 🚀 dotnet/aspnetcore

## 🌟 Trọng tâm
ASP.NET Core là một framework mã nguồn mở, đa nền tảng do Microsoft phát triển để xây dựng các ứng dụng hiện đại kết nối internet trên nền tảng điện toán đám mây.

## 🎯 Bài toán giải quyết & Khách hàng mục tiêu
- **Bài toán giải quyết:** Cung cấp một bộ khung (framework) tối ưu, nhẹ và linh hoạt cho việc phát triển và triển khai đa dạng các loại ứng dụng như ứng dụng web, API, ứng dụng IoT và backend cho thiết bị di động. Nó giải quyết vấn đề hiệu năng, khả năng mở rộng và hạn chế phụ thuộc nền tảng hệ điều hành.
- **Khách hàng mục tiêu:** Các lập trình viên .NET, kiến trúc sư phần mềm, và các doanh nghiệp muốn xây dựng các ứng dụng web và dịch vụ backend hiệu suất cao, có thể chạy linh hoạt trên Windows, macOS, hoặc Linux, đồng thời dễ dàng triển khai lên đám mây (Cloud).

## 🛠️ Công nghệ cốt lõi
- **Ngôn ngữ:** C#, F#
- **Nền tảng:** .NET (chạy đa nền tảng)
- **Kiến trúc:** Modular components (Các thành phần mô-đun)
- **Hệ sinh thái liên quan:** Entity Framework Core (ORM), Razor (UI toolkit), .NET Runtime.

## 💡 Tại sao đáng chú ý?
ASP.NET Core nổi bật nhờ hiệu năng cực cao, thiết kế theo hướng mô-đun giúp tối thiểu hóa tài nguyên tiêu thụ, khả năng chạy đa nền tảng (cross-platform), và sự hỗ trợ mạnh mẽ từ cộng đồng mã nguồn mở cũng như Microsoft. Đây là một trong những framework hàng đầu hiện nay cho việc phát triển web backend, tích hợp dễ dàng với các dịch vụ đám mây và hệ sinh thái .NET khổng lồ.
