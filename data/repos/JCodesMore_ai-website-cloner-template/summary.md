# 🚀 ai-website-cloner-template

## 🌟 Trọng tâm
Mẫu dự án (template) mạnh mẽ hỗ trợ việc sử dụng các tác nhân AI (AI agents) để dịch ngược và sao chép bất kỳ trang web nào thành một dự án Next.js 16 sạch sẽ và hiện đại với chỉ một câu lệnh `/clone-website`.

## 🎯 Bài toán giải quyết & Khách hàng mục tiêu
- **Bài toán:** Khôi phục mã nguồn bị mất, nâng cấp và di chuyển nền tảng từ các hệ thống cũ (WordPress, Webflow) sang mã Next.js mới, hoặc học hỏi cách các trang web thực tế xây dựng bố cục và hiệu ứng.
- **Khách hàng mục tiêu:** Lập trình viên frontend, các nhà phát triển web muốn sao chép nhanh giao diện web (không dùng cho mục đích mạo danh, lừa đảo).

## 🛠️ Công nghệ cốt lõi
- **Next.js 16** (App Router, React 19, TypeScript strict)
- **Tailwind CSS v4** (thiết kế với oklch tokens)
- **shadcn/ui** (dựa trên Radix primitives)
- **AI Coding Agents:** Hỗ trợ rộng rãi các AI như Claude Code, Gemini CLI, Cursor, Windsurf, v.v.

## 💡 Tại sao đáng chú ý?
Tự động hóa hoàn toàn luồng dịch ngược giao diện web bằng cách trích xuất token thiết kế, phân tích tương tác, viết thông số cho component, và sử dụng tác nhân AI song song để xây dựng và kiểm tra giao diện bằng hình ảnh, giúp tiết kiệm hàng chục giờ code thủ công.
