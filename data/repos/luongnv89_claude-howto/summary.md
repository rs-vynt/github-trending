# 🚀 claude-howto

## 🌟 Trọng tâm
Cung cấp hướng dẫn toàn diện, trực quan và dựa trên ví dụ thực tế về Claude Code. Repo bao gồm từ các khái niệm cơ bản đến việc điều phối các agent nâng cao, hooks, skills và MCP servers, cung cấp các template có thể sao chép-dán mang lại giá trị sử dụng ngay lập tức.

## 🎯 Bài toán giải quyết & Khách hàng mục tiêu
- **Bài toán giải quyết**: Việc tài liệu chính thức chỉ mô tả tính năng mà không hướng dẫn cách kết hợp chúng (như slash commands với hooks, memory, và subagents) làm người dùng khó xây dựng luồng công việc hiệu quả. Repo giải quyết điều này bằng cách cung cấp lộ trình học tập rõ ràng cùng các ví dụ ở cấp độ production.
- **Khách hàng mục tiêu**: Các lập trình viên và kỹ sư phần mềm đã cài đặt Claude Code và muốn tối đa hóa hiệu suất làm việc bằng cách thiết lập tự động hóa, quy trình review code, hoặc hệ thống AI agent phức tạp.

## 🛠️ Công nghệ cốt lõi
- Claude Code (v2.1+)
- Markdown & Mermaid diagrams (để trực quan hóa kiến trúc)
- Model Context Protocol (MCP)
- Shell Scripting (cho Hooks)
- Python (cho kiểm thử, sinh EPUB ebook)

## 💡 Tại sao đáng chú ý?
- **Đạt top #1 GitHub Trending**: Được cộng đồng lập trình viên đón nhận và sử dụng rộng rãi (hơn 36.000 sao).
- **Học tập có lộ trình**: Cung cấp bài kiểm tra tự đánh giá và lộ trình học tập từ 15 phút (cơ bản) đến khóa học hoàn chỉnh cuối tuần (11-13 giờ).
- **Tính thực tiễn cao**: Chứa các cấu hình copy-paste để thiết lập nhanh chóng slash commands, plugins, subagents, và memory cho mọi dự án.
- **Cập nhật liên tục**: Luôn đồng bộ với các phiên bản Claude Code mới nhất (v2.1.160) và hỗ trợ các mô hình AI mới như Claude Sonnet 4.6, Opus 4.8.
