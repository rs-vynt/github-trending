goose là một AI agent đa năng chạy trên máy của bạn. Không chỉ dành cho code — hãy sử dụng nó để nghiên cứu, viết lách, tự động hóa, phân tích dữ liệu hoặc bất cứ việc gì bạn cần hoàn thành.

Một ứng dụng desktop nguyên bản cho macOS, Linux và Windows. Một CLI đầy đủ cho các quy trình làm việc trên terminal. Một API để nhúng nó vào bất cứ đâu. Được xây dựng bằng Rust để đạt hiệu suất và khả năng tương thích cao.

goose hoạt động với hơn 15 nhà cung cấp — Anthropic, OpenAI, Google, Ollama, OpenRouter, Azure, Bedrock, và nhiều hơn nữa. Sử dụng API key hoặc các gói đăng ký Claude, ChatGPT, hay Gemini hiện có của bạn thông qua [ACP](https://goose-docs.ai/docs/guides/acp-providers). Kết nối với hơn 70 tiện ích mở rộng thông qua tiêu chuẩn mở [Model Context Protocol](https://modelcontextprotocol.io/).

goose là một phần của [Agentic AI Foundation (AAIF)](https://aaif.io/) tại Linux Foundation.

# Bắt đầu

**[Tải xuống ứng dụng desktop](https://goose-docs.ai/docs/getting-started/installation)** cho macOS, Linux, và Windows.

Hoặc cài đặt CLI:

```bash
curl -fsSL https://github.com/aaif-goose/goose/releases/download/stable/download_cli.sh | bash
```
