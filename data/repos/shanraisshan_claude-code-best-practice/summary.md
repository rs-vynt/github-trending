# 🚀 claude-code-best-practice
## 🌟 Trọng tâm
Dịch chuyển từ "vibe coding" (lập trình theo cảm hứng) sang "agentic engineering" (kỹ nghệ tác tử) - thực hành giúp Claude trở nên hoàn hảo.
## 🎯 Bài toán giải quyết & Khách hàng mục tiêu
**Bài toán giải quyết:** Hướng dẫn cách sử dụng công cụ Claude Code hiệu quả nhất, tận dụng tối đa các tính năng như subagents (tác tử phụ), commands (lệnh), skills (kỹ năng), bộ nhớ, hooks và MCP để tối ưu hóa quy trình tự động hóa phát triển phần mềm và điều phối đa tác tử.
**Khách hàng mục tiêu:** Lập trình viên, kỹ sư phần mềm, kỹ sư AI, và các đội ngũ công nghệ muốn làm chủ Claude Code để chuyển đổi quy trình làm việc từ việc nhắc lệnh đơn giản (prompting) sang xây dựng hệ thống kỹ nghệ tác tử chuyên nghiệp.
## 🛠️ Công nghệ cốt lõi
- Claude Code CLI
- AI Agents (Tác tử AI) & Subagents (Tác tử phụ)
- Slash Commands (Các lệnh gạch chéo)
- Skills (Kỹ năng) & Hooks
- MCP (Model Context Protocol) Servers
- Orchestration Workflows (Luồng công việc điều phối)
- Git Worktrees
## 💡 Tại sao đáng chú ý?
Kho lưu trữ này thu hút gần 60.000 sao trên GitHub và đứng đầu danh sách GitHub Trending nhờ vào việc tổng hợp một cách đồ sộ và chi tiết hơn 83 mẹo, thủ thuật thực tiễn từ chính những người tạo ra Claude (như Boris Cherny, Thariq) và cộng đồng. Nó không chỉ là lý thuyết mà đi kèm với các bản triển khai thực tế, ví dụ mẫu và luồng công việc giúp tiết kiệm thời gian, giảm lỗi và tăng năng suất mã hóa bằng AI lên gấp nhiều lần.
