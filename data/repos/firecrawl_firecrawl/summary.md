# 🚀 firecrawl/firecrawl
## 🌟 Trọng tâm
Firecrawl là một API cung cấp khả năng tìm kiếm, cào dữ liệu (scrape), và tương tác với các trang web ở quy mô lớn, giúp chuyển đổi nội dung web thành dữ liệu Markdown sạch hoặc JSON có cấu trúc sẵn sàng cho các mô hình ngôn ngữ lớn (LLM) và các ứng dụng AI.

## 🎯 Bài toán giải quyết & Khách hàng mục tiêu
- **Bài toán giải quyết:** Việc thu thập dữ liệu web thường gặp nhiều khó khăn do trang web sử dụng nhiều JavaScript, bị chặn nội dung, cần quản lý proxy xoay vòng, và HTML chứa nhiều mã rác khó dùng cho LLM. Firecrawl tự động hóa các rào cản này và trích xuất ra dữ liệu chất lượng cao.
- **Khách hàng mục tiêu:** Các nhà phát triển AI, kỹ sư dữ liệu, và các hệ thống tác nhân tự trị (AI agents) cần tích hợp dữ liệu web thời gian thực một cách dễ dàng và đáng tin cậy.

## 🛠️ Công nghệ cốt lõi
- Khả năng xử lý và phân tích nội dung từ các trang web phức tạp (JS-heavy), kể cả các tài liệu như PDF, DOCX lưu trữ trên web.
- Tích hợp tính năng "Agent": Tự động tìm kiếm, cuộn trang, nhấp chuột và thu thập dữ liệu chỉ dựa trên prompt (mô tả yêu cầu) mà không cần lập trình URL cụ thể.
- Hệ thống API linh hoạt hỗ trợ nhiều tính năng (Search, Scrape, Map, Crawl, Batch Scrape) cùng hệ sinh thái SDK đa dạng (Python, Node.js, Java, Rust, v.v.).

## 💡 Tại sao đáng chú ý?
Firecrawl đạt độ bao phủ lên tới 96% các trang web, đi kèm tốc độ truy xuất cực nhanh (độ trễ P95 chỉ 3.4 giây). Là một dự án mã nguồn mở mạnh mẽ (đạt hơn 130K sao trên GitHub), nó giúp các nhà phát triển xây dựng AI app tiết kiệm đáng kể số lượng token nhờ đầu ra được làm sạch, đồng thời đơn giản hóa tối đa việc thu thập dữ liệu cho bất kỳ AI agent nào.
